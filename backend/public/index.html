About CrowdDot

CrowdDot is a crypto community game where every player moves a shared blue “dot” by a small increment. Work together to navigate the maze!

  • Contract Address (CA): Only released on @CrowdDot on Twitter. It will update here within 1-2 minutes. The page also may need to be refreshed.
  • Live Market Data: Provided by Pump Portal API.
  • Dev Wallet: Starts with 2 SOL (~5% supply). Hitting a red wall triggers a sale of 5% of the dev wallet (5-10s delay, game pauses 5s).
  • Reach the green circle: 10% of the dev wallet is locked for 2 weeks via Streamflow. A tweet with a screenshot and link is automatically posted for transparency.
  • Levels: 7 total, each harder than the last. Next level starts 20s after a win.
  • Future: If successful, the game will expand with new features and modes. After 2 weeks, V2 will launch under the same CA.
  • Github: No public repo yet due to issues with server security, but one will be released post-launch for feedback and contributions.

This is a small scale project with a small team. Expect bugs or inconsistencies. Thanks for playing and helping test CrowdDot!
