About CrowdDot

CrowdDot is a collaborative online game where players work together in real time to control a single dot on the screen. The game is designed to showcase the power of community coordination and teamwork.

  • Players collectively control a single "dot" on the game board.
  • Each player can vote to move the dot every few seconds.
  • If the dot hits a red wall, the level restarts and the group tries again.
  • If the dot reaches the green circle, the group advances to the next level.
  • The game demonstrates the fun and challenge of real-time community coordination.

This project showcases how real-time communication and collective decision-making can create unique, interactive experiences.
